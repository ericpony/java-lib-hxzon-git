本书不仅介绍了CSS基础知识，而且全面细致地讲解了布局与效果，最后展望了HTML5和CSS3的未来，是构建现代Web布局的实用指南。
本书作者Eric A. Meyer是一位国际公认的HTML、CSS、Web标准方面的专家，他在本书中深入浅出地分析了普遍适用的实用技术，包括如何选择正确的工具。

本书主要内容如下：
 显示/隐藏元素；
 用XHTML创建背景；
 15个布局技术，包括“清除浮动”（clearfix）和Holy Grail技术，以及应用两栏/三栏布局、伪造栏(faux column)布局、“唯一布局”（One True Layout）、基于em的布局、流式栅格等技术；
 各种CSS效果，包括CSS弹出窗口、框冲切（boxpunching）、圆角、CSS sprite、滑动门（Sliding Door）、流式漂白布局（Liquid Bleach）、参差浮动（ragged float）等；
 为表添加样式，内容涉及thead、 tfoot、tbody、行标题（row header）、面向列的样式设计（ column-oriented styling ）等；
 各种CSS3元素，内容涉及设置圆角、多背景、 RGBA，以及通过 jQuery执行CSS3 选取操作等。

本书适合具有一定的CSS和JavaScript使用经验且正在寻求更先进技术的开发人员学习参考。
