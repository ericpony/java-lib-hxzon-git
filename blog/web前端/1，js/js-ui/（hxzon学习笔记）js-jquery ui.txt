（hxzon学习笔记）js-jquery ui


1，

暂时没有树，表格等控件。

--
交互类：

可拖动（移动），可放置（接收），可调整尺寸，可选中，可移动排序。

--
控件类：

手风琴（Accordion，切换展开），自动填充，按钮，日期选中器，
对话框（阻塞，移动，调整尺寸，显示或隐藏），
菜单，进度条，
滑动条（微调），微调器spinner，标签页（切换显示），悬浮提示。

--
效果类：

addClass/removeClass（添加或移除css类，渐变），
toggleClass（添加或移除css类），

switchClass（切换css类），

animate（渐变，如颜色渐变），
effect（用于显示或隐藏的各类特效），
show/hide（显示或隐藏），
toggle（切换显示或隐藏）。

--
工具类：

位置，控件工厂。
