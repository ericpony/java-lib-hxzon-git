在开发中，如何进行类型的选择。
目录
当你知道值不可能为负时，请使用无符号类型。
请使用 int 进行整数运算。通常情况下，short 太小。然而实际情况下，long 往往和 int 一样大。如果你的数值大于 int的最小大小，请使用 long long。
不要在算数表达式中使用普通的 char 或 bool。只将他们用于存储字符和布尔值。使用 char 进行运算将带来问题，因为在某些机器上 char 是有符号的，而在另外一些机器上 char 是无符号的。如果你需要使用一个很小的整型，请显式指定 signed char 或者 unsigned char。
使用 double 进行浮点运算。通常，float 没有足够的精度，并且双精度计算与单精度计算的成本是微不足道的。事实上，在某些机器上，双精度计算还要快于单精度计算。long double 提供的精度通常是不需要的，而且经常需要考虑运行时成本。
