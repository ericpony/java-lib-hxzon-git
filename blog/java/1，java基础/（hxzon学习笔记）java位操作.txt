2，java比较

Collections.sort为升序。

如果本对象要比参数对象靠前，则应返回-1。

public int compareTo(Integer anotherInteger) {
    return compare(this.value, anotherInteger.value);
}

public static int compare(int x, int y) {
    return (x < y) ? -1 : ((x == y) ? 0 : 1);
}

逆序，即大数在前：
public int compareTo(int o)
{
    return o - this;
}
