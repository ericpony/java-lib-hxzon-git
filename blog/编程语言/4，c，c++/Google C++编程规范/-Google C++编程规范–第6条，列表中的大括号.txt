==

【规范】

可以的话，放在一行；否则，从左大括号开始换行。

【详细解释一下】

尽可能将所有内容都放在一行内，如果超过行宽限制，那么应该确保左大括号是一行的最末尾字符，而右大括号是一行的首字符。

MyType m = {
  superlongvariablename1,
  superlongvariablename2,
  {short, interior, list},
  {
    interiorwrappinglist,
    interiorwrappinglist2
  }
};

这个例子就很清晰的解释了我们今天的规则。
